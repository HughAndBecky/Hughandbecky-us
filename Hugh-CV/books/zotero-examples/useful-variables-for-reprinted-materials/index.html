Zotero Fields

CSL Value Description
Original Date The original date an item was published. Enter in ISO format (year-month-day).
Original Title The original title of a work (e.g., the untranslated title).
Original Publisher The publisher of the original version of an item (e.g., the untranslated version).
Original Publisher Place The geographic location of the publisher of the original version of an item (e.g., the untranslated version).
Original Author A type of Creator. The original creator of a work.
